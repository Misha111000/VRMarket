Meta Quest 3

The Meta Quest 3, released in 2023, represents the latest in standalone VR technology. It boasts enhanced performance with a new Snapdragon XR2+ processor, 12GB of RAM, and significantly improved graphics with 2064 x 2208 pixels per eye. The headset features a slimmer and more comfortable design, improved hand tracking, and advanced mixed reality capabilities. With a broad array of applications spanning gaming, fitness, and productivity, the Meta Quest 3 continues to push the boundaries of immersive experiences.

INFORMATION

This headset has the following:

Core performance and architecture

SoC : Snapdragon XR2 Gen1

DRAM : 8GB

Base storage : 128GB, 512GB

Ergonomics

Strap/ergo : Soft strap, hard strap available separately

Levels of immersion : One level (fully immersed)

Optics technology

Display resolution (per eye) : 2064 x 2208 | 1218 PPI | 25 PPD

Refresh rate : 72Hz, 80 Hz, 90Hz, 120Hz

FOV (degrees) : 110 horizontal / 96 vertical

Optics : Pancake lens, continuous IAD

Color gamut : 100% of sRGB

Headset tracking

Face/eye tracking : No

Hand tracking : Yes

Controllers

Controller tracking : Hybrid using computer vision and machine learning tracking

Haptics : TruTouch variable haptics

Rechargeable Battery : No

Passthrough fidelity

Color (4MP, 19 PPD)

Charging

Charging cord : Included

Charging dock : Available from third-party vendors

Battery : : 5060 mAh* - 2.2 hour runtime*
